## Deck-Level Focus

Defines what visualization is, why it matters, and how it supports reasoning, context building, and decision making.

## Core Knowledge

- Visualization turns structured data into visual representations that amplify cognition.
- Good visualization supports analysis, pattern discovery, communication, and decision making.
- Information visualization focuses on abstract data, while scientific visualization focuses on
    physical or simulated phenomena.
- Visualization value comes from seeing data in context, not only from making graphics attractive.
